Youtube Downloader

C'est un projet que j'ai créé afin de répondre à un problème :
Comment télécharger une musique ou une vidéo en étant sûr de ne pas avoir de virus et avec la meilleure qualité audio et vidéo disponible ?

J'ai donc créé mon propre convertisseur YouTube pour télécharger des vidéos présentes sur YouTube.

Le backend est en Python, réalisé grâce au framework Flask. Ce projet est open source sur mon GitHub dans le même dépôt que ce site.

Pour l'hébergement, j'ai décidé de l'héberger sur mon Raspberry Pi. Cela a été rendu possible grâce à un "tunnel Cloudflare", qui me permet de me connecter à mon petit serveur sans avoir à ouvrir aucun port de ma box.

Youtube Downloader

This is a project that I created to solve a problem:
How to download a music or a video while being sure to not have any virus and with the best audio and video quality available ?

So I created my own YouTube converter to download videos from YouTube.

The backend is in Python, made with the Flask framework. This project is open source on my GitHub in the same repository as this website.

For the hosting, I decided to host it on my Raspberry Pi. This was made possible thanks to a "Cloudflare tunnel", which allows me to connect to my little server without having to open any port of my box.

github.com/PaulBayfield/bayfield.dev
